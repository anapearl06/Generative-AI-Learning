# вҡҷпёҸ Getting Started

## 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/anapearl06/Generative-AI-Learning.git
```

---

## 2пёҸвғЈ Navigate to the Project Folder

Example:

```bash
cd Generative-AI-Learning/week1/day5
```

or

```bash
cd Generative-AI-Learning/week2/day6
```

---

## 3пёҸвғЈ Install Dependencies

```bash
uv sync
```

This command automatically:

- Creates a virtual environment
- Installs all dependencies
- Uses versions locked in `uv.lock`

---

## 4пёҸвғЈ Configure Environment Variables

Create a `.env` file.

```env
GROQ_API_KEY=your_groq_api_key
```

Never upload your API keys to GitHub.

---

## 5пёҸвғЈ Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

### macOS / Linux

```bash
source .venv/bin/activate
```

---

## 6пёҸвғЈ Run Any Project

Example:

```bash
python resume_parser.py
```

or

```bash
python prompt_eng.py
```

Each project contains its own implementation and examples.

---
